# Vortex

Vortex is a full-stack social media web application inspired by Instagram.  
Users can create accounts, log in securely, upload posts, and interact with content through a personalized feed.

🔗 Live App: https://vortex-blue-eight.vercel.app/

---

## Features

- User authentication (signup & login)
- Secure password hashing
- Create and view posts
- Image-based feed
- Like functionality
- Protected routes
- Responsive UI

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication
- Bcrypt

### Database
- MongoDB (Mongoose)

### Deployment
- Vercel (Frontend)
- Cloud-hosted Node/Express server
